## Автоматические UI‑тесты для сайта SauceDemo

Этот небольшой проект содержит примеры автоматизированных UI‑тестов на Python для сайта интернет‑магазина [SauceDemo](https://www.saucedemo.com).  
Тесты написаны с использованием `pytest`, `playwright` и плагина `pytest-playwright`.

### Что проверяют тесты

- **Авторизация (`tests/test_login.py`)**
  - **Успешный вход** стандартного пользователя и переход на страницу со списком товаров.
  - **Попытка входа заблокированного пользователя** — отображается сообщение об ошибке.
  - **Неверные учётные данные** — отображается ошибка о неправильном логине или пароле.

- **Корзина и оформление заказа (`tests/test_cart_and_checkout.py`)**
  - **Добавление товара в корзину** — счётчик на иконке корзины и наличие товара внутри корзины.
  - **Удаление товара из корзины** — после удаления остаётся только ожидаемый товар.
  - **Полный сценарий оформления заказа** — добавление товара, заполнение формы (имя, фамилия, индекс) и успешное завершение покупки.

### Стек и зависимости

- **Язык**: Python 3.10+ (рекомендуется актуальная версия Python 3)
- **Тестовый фреймворк**: `pytest`
- **UI‑автоматизация**: `playwright`
- **Интеграция pytest + Playwright**: `pytest-playwright`

Зависимости перечислены в файле `requirements.txt`.

### Подготовка окружения

1. **Клонировать репозиторий или скопировать проект** в любую папку.
2. **Создать и активировать виртуальное окружение (рекомендуется)**.

   Пример для Windows (PowerShell):

   ```powershell
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   ```

   Пример для bash (Git Bash, WSL и т.п.):

   ```bash
   python -m venv venv
   source venv/Scripts/activate
   ```

3. **Установить зависимости**:

   ```bash
   pip install -r requirements.txt
   ```

4. **Установить браузеры для Playwright** (один раз):

   ```bash
   playwright install
   ```

### Запуск тестов

Находясь в корне проекта (`ml_tests`), выполнить:

```bash
pytest
```

По умолчанию будут запущены все тесты в папке `tests`.

Примеры полезных команд:

- **Запуск только тестов логина**:

  ```bash
  pytest tests/test_login.py
  ```

- **Запуск только тестов корзины и оформления заказа**:

  ```bash
  pytest tests/test_cart_and_checkout.py
  ```

### Структура проекта (основные файлы)

- **`tests/test_login.py`** — тесты авторизации (успешный вход, заблокированный пользователь, неверные данные).
- **`tests/test_cart_and_checkout.py`** — тесты корзины и полного процесса оформления заказа.
- **`requirements.txt`** — список Python‑зависимостей для запуска тестов.

### Примечания

- В тестах используется демонстрационный стенд SauceDemo, доступный по адресу `https://www.saucedemo.com/`.
- Учётные данные (`standard_user`, `locked_out_user`, `secret_sauce`) — стандартные тестовые пользователи из документации SauceDemo.
